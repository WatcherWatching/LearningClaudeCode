  learning/educational project for students building a full-stack web app.

  Project Structure:

  - Database: SQLite (via database/db.py)
  - Frontend: HTML templates + CSS/JS

  Current State:
  - Landing, register, and login pages are set up
  - Database module is a stub - students need to implement get_db(), init_db(),
  and seed_db()

    - /logout - Step 3
    - /profile - Step 4
    - /expenses/add - Step 7
    - /expenses/edit - Step 8
    - /expenses/delete - Step 9

  What it's meant to do:
  A web app where users can register, log in, and manage their personal expenses

  step-by-step tutorial where students progressively build features.
